① In-situ Analysis Pipeline

Spectra processing Peak tracking Export & reporting

用於 in-situ UV-Vis/NIR 量測（溶劑交換、乾燥、反射峰移動等），將原始資料自動化整理， 並同步輸出關鍵指標（最大反射率、峰值波長、時間序列）。

  • 自動讀取量測檔案、整理時間軸與取樣點
  • 峰值偵測 / peak tracking（可用於監控 equilibration）
  • 輸出：圖 + summary table（CSV/Excel）便於放入報告/論文
  • 可擴充：baseline correction、平滑、批次處理、多樣品比較

② Bruggeman Refractive Index Calculator

EMA model neff prediction GUI tool

以 Bruggeman effective medium approximation 估算複合材料或多孔結構的有效折射率 neff， 輔助材料配方、孔隙率/填充率推估與光學設計（如 Bragg 反射波長預測的前處理）。

  • 輸入：各相折射率、體積分率（或孔隙率/填充率）
  • 輸出：neff、敏感度掃描（f vs neff）
  • 可用於：多孔薄膜、BCP 模板填充、MOF/氧化物 backfill 等
  • 支援：快速試算、批次參數 sweep、結果匯出

Workflow

Measurement → Analysis → Model → Decision

In-situ Analysis

  • 資料匯入：原始光譜/反射率檔案（時間序列）
  • 前處理：對齊時間、選波段、可選平滑/基線
  • 特徵萃取：peak λ、peak intensity、統計指標
  • 輸出：figures + tables → 報告/簡報可直接使用

Bruggeman Calculator

  • 定義材料相：n、volume fraction / porosity
  • 求解 neff（支援參數 sweep）
  • 與實驗結果交叉驗證：反射峰位置/顏色漂移
  • 作為實驗設計工具：快速縮小配方與條件範圍
